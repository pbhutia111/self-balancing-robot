# CAD Files

3D models for the robot's printed/cut parts.

Put files here, for example:
- `chassis.stl` — printable mesh
- `chassis.f3d` / `.step` / `.scad` — editable source (so others can modify)
- `motor-mount.stl`, `battery-tray.stl`, etc.

**Tip:** include both the editable source (`.step`, `.f3d`, `.scad`, `.FCStd`) *and* the
ready-to-print `.stl`. `.step` is the most universal editable format if you want anyone to
be able to open it.

| Part | File | Notes |
|------|------|-------|
| _Chassis_ | _chassis.stl_ | _print settings, material..._ |
